# IETF OAuth Working Group

The **IETF OAuth Working Group** is the Internet Engineering Task Force body that standardizes the OAuth protocol family. It produces the RFCs and drafts the MCP Authorization profile depends on.

## Outputs referenced by MCP Authorization

The [[mcp-authorization-overview]] builds on OAuth WG work, including:

- [[oauth-2-1|OAuth 2.1]] (draft) and [[rfc-7636-pkce|PKCE (RFC 7636)]]
- [[rfc-6750-bearer-token-usage|Bearer Token Usage (RFC 6750)]]
- [[rfc-8414-authorization-server-metadata|AS Metadata (RFC 8414)]] and [[rfc-9728-protected-resource-metadata|Protected Resource Metadata (RFC 9728)]]
- [[rfc-7591-dynamic-client-registration|Dynamic Client Registration (RFC 7591)]]
- [[rfc-8707-resource-indicators|Resource Indicators (RFC 8707)]]
- [[rfc-9207-authorization-server-issuer-identification|Issuer Identification (RFC 9207)]]
- [[oauth-client-id-metadata-documents|Client ID Metadata Documents (draft)]]

IETF RFCs and active OAuth WG drafts are **high**-confidence sources in this wiki.
