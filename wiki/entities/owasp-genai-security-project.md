# OWASP GenAI Security Project

The **OWASP GenAI Security Project** (genai.owasp.org) is the OWASP project that produces the [[owasp-llm-top-10|OWASP Top 10 for LLM Applications]], the community-driven risk list this wiki treats as a high-confidence source. The 2025 release names Steve Wilson as project lead and Ads Dawson as technical lead and vulnerability entries lead; the list is assembled from a worldwide contributor base via brainstorming, voting, and practitioner feedback ([[owasp-llm-top-10-2025]], *Letter from the Project Leads*, *Moving Forward*).

The project states a vendor-neutral, unbiased approach: sponsors (2025 gold sponsors include HiddenLayer, Mend.io, Palo Alto Networks, and Snyk) "do not receive special governance considerations," and its documents are published under CC BY-SA 4.0.
